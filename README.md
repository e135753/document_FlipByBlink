短めのAbstract
--------
**わずかに長い瞬きでページめくりする電子書籍UIの提案をする。** 画像に示すように閉瞼を少し長めに行うとページ送りを行い、目を開けた際には次のページに遷移しているUI。

比較的長時間の瞬目は日常的な動作である故に自然かつ快適に扱えることを期待。普及している一般的なUIに追加される形。障害者向けというわけではなく、広く一般的なUIとして。提案する手法は「固定レイアウト」の電子書籍のみを対象。例えば紙本をそのまま転用したものや漫画。

デモアプリを作成し予備実験を行なったところ、概ね高評価を得た。
